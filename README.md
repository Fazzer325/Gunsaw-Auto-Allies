# Gunsaw-Auto-Allies
BepInEx 5 mod for Gunsaw that adds configurable AI allies, formations, rescue, markers, and manual orders.

Download .dll
https://github.com/Fazzer325/Gunsaw-Auto-Allies/releases/tag/Mod

This mod spawns configurable AI allies at level start and gives them follow, combat, formation, rescue, collision, marker, and manual order controls.

Author
Created by Fazzer with AI-assisted development using OpenAI Codex.

Features
Configurable ally count per level.
Behavior modes: Follow/Combat, Defensive, Follow only.
Formations: Close, Line, Behind, Spread.
Manual orders with configurable keys.
Ally rescue when they are left behind or unloaded by distance.
Optional ally markers.
Configurable player/ally collision.
Ally bullets can ignore the player.
Cursor aim modes: Never, On fire, Always.
Default Controls
Open menu: .
Attack point order: G
Hold position toggle: H
Keys can be changed from the in-game menu or the BepInEx config file.

Requirements
Gunsaw
BepInEx 5
